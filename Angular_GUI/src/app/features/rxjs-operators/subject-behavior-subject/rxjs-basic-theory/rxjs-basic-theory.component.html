What is a Subject?

In RxJS, a Subject is a special type of Observable that allows values to be multicasted to multiple Observers. Unlike a regular Observable, a Subject is also an Observer.
means you can subscribe to it and also emit values through it, making it a powerful tool for event handling, inter-component communication, and state management in Angular applications.

Multicasting: A Subject allows multiple observers to listen to the same Observable execution. When a value is emitted, all subscribers receive the value.

Types of Subjects

Multicasting: A basic Subject that does not hold any values or have any special behavior.

BehaviorSubject: Stores the latest value emitted to its consumers and immediately emits this value to new subscribers.

ReplaySubject: Stores a specified number of previous values and emits them to new subscribers.

AsyncSubject: Emits the last value (and only the last value) to its observers when the execution completes.
