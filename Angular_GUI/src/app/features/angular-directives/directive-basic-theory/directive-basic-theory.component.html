What is Directives?

Components break up the application into smaller parts; whereas, Directives add behavior to an existing DOM element.

Type of Directives?
  • Component Directives are the most common directives. They are directives with a template.
  • Structural Directives change the DOM layout by adding and removing DOM elements. Ex- ngFor, ngIf, ngSwitch.
  • Attribute directives change the appearance or behavior of an element, component, or another directive. Ex- ngClass, ngStyle.
